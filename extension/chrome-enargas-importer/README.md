# Registro GNC - Importador ENARGAS

Extension local de Chrome para enviar el texto visible del detalle de ENARGAS a la planilla Registro GNC.

## Instalacion

1. Abrir Chrome.
2. Ir a `chrome://extensions`.
3. Activar `Modo desarrollador`.
4. Elegir `Cargar descomprimida`.
5. Seleccionar esta carpeta: `extension/chrome-enargas-importer`.
6. Si la planilla se abre como archivo local (`file://`), entrar a los detalles de la extension y activar `Permitir acceso a URL de archivo`.

## Uso

1. Abrir Registro GNC en una pestaña.
2. Abrir ENARGAS y entrar al detalle de la operacion.
3. Tocar la extension `Registro GNC - Importador ENARGAS`.
4. Elegir `Enviar a planilla`.
5. Revisar los campos importados antes de guardar.
